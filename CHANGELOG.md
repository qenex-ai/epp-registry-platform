# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Initial public release
- EPP Server implementing RFC 5730-5734
- RDAP Server implementing RFC 7480-7484
- WHOIS Server implementing RFC 3912
- Admin Dashboard for registry management
- PostgreSQL database schema
- Docker Compose deployment
- Kubernetes manifests
- Comprehensive documentation

## [1.0.0] - 2025-11-19

### Added
- Complete EPP protocol implementation
- Domain, contact, and host object support
- DNSSEC support
- Multi-TLD capability
- RESTful RDAP API
- Traditional WHOIS protocol
- Web-based administration interface
- Production-ready database schema
- Docker containerization
- Kubernetes deployment manifests
- MIT License
